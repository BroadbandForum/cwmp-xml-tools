TRminator Command-Line Interface

Introduction

The following instructions are for the TRminator Command-Line Interface.
If you are interested in the graphical interface of TRminator, go here.

Directions

Running the program: To invoke the program in CLI mode, run it from a command prompt, with the "-cli" argument: 
user@host> java -jar TRminator.jar -cli [arguments]
There are 2 required arguments:
     "-i  [path]" is the input option.   It is to be followed by a string representing the path to the input file.
     "-o [path]" is the output option. It is to be followed by a string representing the path to the output file.

Options/Arguments:

Other than the mandatory arguments ("-cli" "-i [path]" and "-o [path]"), there are these optional arguments.
Any of these optional arguments can be applied in no particular order:
     "-nodiff" -- turn off HTML formatting that shows difference between current and previous version of document.
     "-showprofiles" -- turn on insertion of a list of profiles that were discovered while making the table.
     "-looks" -- turn on formatting for fitting table into standard document/browser window.
