The Table Class

Package: threepio.tabler.container
Description: The threepio.tabler.container.Table class is an IndexedHashMap that has String keys and row values
Requirements: none
Uses:
Multiple derivatives of Table are used to store the information found in XML documents.
All Tabler objects produce at least one of these derivatives.

The Row Class

Package: threepio.tabler.container
Description: The threepio.tabler.container.Row class is an ArrayList of StringCells With a static size.
Requirements: No adding is done, other than during construction.
Uses:
Rows are used to model a row for a table for organization and printing.

The StringCell Class

Package: threepio.tabler.container
Description: The threepio.tabler.container.StringCell class is an Object that models a cell in a table. Requirements: none
Uses:
StringCells store the actual data in the cell, as well as attributes that describe them.
There is a "flag" boolean value available, to signify a need for advanced processing.
For example, the flag is used in the current implementation of Tabler to tell the methods if new information is to be appended to the current data, or is to replace it.
