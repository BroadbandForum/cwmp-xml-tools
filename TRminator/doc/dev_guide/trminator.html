About TRminator

TRminator is an add-on package for the Threepio XML Data Model table-maker engine.
It consists of a graphical user interface and a command-line user interface, as well as a Post Processor for Tables created by Threepio that processes BBF-style markup.

The User Interfaces

TRminator provides a graphical user interface (defined in GUI.java) and a command-line user interface (defined in CLI.java).
The GUI contains a mode for non-BBF documents, but the CLI is currently optimized for BBF documents.

Table Post Processor

TRminator contains a Table Post Processor that handles BBF-style markup in curly braces. It is defined in "TablePostProcessor.java."
There are Objects of class MarkupProcessor that are also defined in this file. They are used one after another, in the method doReplaces.
Some of these may have to be run before each other, if the markup handling of one creates more markup, or markup depends on data created by other markup.
A ModelTable object created using Threepio can be run through TablePostProcessor's deMarkupTable method to remove and process BBF markup.
This needs to be done prior to printing the table out.
