About TRminator

Purpose:

TRminator was created for the Broadband Forum and all users of their Data Models, specifically the XML documents used for CWMP/TR-069.
TRminator enables the user to create human-readable representations of the information in the XML Data Model files.

TRminator was created using Java 6, and requires the Java Runtime Environment to run.
